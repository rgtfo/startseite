# RG/TFO Startseite
Dies ist der SOURCE-CODE der Startseite für das Realgymnasium und die Technologische Fachoberschule (RGTFO) in Meran. Die Website bietet ein zentrales Startmenü, das es Nutzern ermöglicht, schnell zu den wichtigsten Plattformen und Diensten zu navigieren. Sie dient als schnelle und benutzerfreundliche Einstiegshilfe für Schüler, Lehrer und Eltern.
